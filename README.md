# Float Income Simulator

Simulador de retorno por inversión local durante el período de float.

**Brasil (BRL) · México (MXN) · USD**

## Deploy en Streamlit Community Cloud

1. Subir este repo a GitHub
2. Ir a [share.streamlit.io](https://share.streamlit.io)
3. Login con GitHub → New app → seleccionar repo → `app.py` → Deploy

## Correr localmente

```bash
pip install -r requirements.txt
streamlit run app.py
```
